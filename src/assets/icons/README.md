# Icons Directory

This directory contains SVG icon assets for the Fellas Barbershop application.

All icons are implemented inline in the `FIcon.vue` component for optimal performance and flexibility.

## Available Icons

- checkmark
- chevron-down
- chevron-left
- chevron-right
- chevron-up
- close
- menu
- star
- star-filled
- instagram
- phone
- email
- location
- clock
- scissors
- crown

## Usage

```vue
<FIcon name="checkmark" :size="24" color="#FF4458" />
```
